```
# Вариант 7

Минимизировать $f(x,y)=(x-2)^2+(y+3)^2$ при

$$
g_1(x,y)=3x+2y-4=0,\quad g_2(x,y)=x-y-1=0.
$$
```

### Решение, полученное от БЯМ

**Идентификация задачи:**

Задача представляет собой задачу условной оптимизации с двумя линейными ограничениями-равенствами. Целевая функция является квадратичной и строго выпуклой (сумма квадратов с положительными коэффициентами).

**Решение методом множителей Лагранжа:**

**Шаг 1: Составление функции Лагранжа**

Функция Лагранжа имеет вид:

$$L(x, y, \lambda_1, \lambda_2) = (x-2)^2 + (y+3)^2 + \lambda_1(3x + 2y - 4) + \lambda_2(x - y - 1)$$

**Шаг 2: Нахождение частных производных и составление системы уравнений**

Найдем частные производные функции Лагранжа:

$$\frac{\partial L}{\partial x} = 2(x-2) + 3\lambda_1 + \lambda_2 = 2x - 4 + 3\lambda_1 + \lambda_2 = 0$$

$$\frac{\partial L}{\partial y} = 2(y+3) + 2\lambda_1 - \lambda_2 = 2y + 6 + 2\lambda_1 - \lambda_2 = 0$$

$$\frac{\partial L}{\partial \lambda_1} = 3x + 2y - 4 = 0$$

$$\frac{\partial L}{\partial \lambda_2} = x - y - 1 = 0$$

Получаем систему уравнений:

$$
\begin{cases}
2x + 3\lambda_1 + \lambda_2 = 4 \\
2y + 2\lambda_1 - \lambda_2 = -6 \\
3x + 2y = 4 \\
x - y = 1
\end{cases}
$$

**Шаг 3: Решение системы уравнений**

Из четвертого уравнения: $x = y + 1$

Подставляем в третье уравнение:
$$3(y + 1) + 2y = 4$$
$$3y + 3 + 2y = 4$$
$$5y = 1$$
$$y = \frac{1}{5} = 0.2$$

Тогда:
$$x = 0.2 + 1 = 1.2$$

Подставляем найденные значения в первые два уравнения:

Из первого уравнения:
$$2 \cdot 1.2 + 3\lambda_1 + \lambda_2 = 4$$
$$2.4 + 3\lambda_1 + \lambda_2 = 4$$
$$3\lambda_1 + \lambda_2 = 1.6$$

Из второго уравнения:
$$2 \cdot 0.2 + 2\lambda_1 - \lambda_2 = -6$$
$$0.4 + 2\lambda_1 - \lambda_2 = -6$$
$$2\lambda_1 - \lambda_2 = -6.4$$

Решаем систему для множителей Лагранжа:

$$
\begin{cases}
3\lambda_1 + \lambda_2 = 1.6 \\
2\lambda_1 - \lambda_2 = -6.4
\end{cases}
$$

Складываем уравнения:
$$5\lambda_1 = -4.8$$
$$\lambda_1 = -0.96$$

Подставляем в первое уравнение:
$$3(-0.96) + \lambda_2 = 1.6$$
$$-2.88 + \lambda_2 = 1.6$$
$$\lambda_2 = 4.48$$

**Шаг 4: Вычисление значения целевой функции**

$$f(1.2, 0.2) = (1.2-2)^2 + (0.2+3)^2 = (-0.8)^2 + (3.2)^2 = 0.64 + 10.24 = 10.88$$

**Шаг 5: Проверка критерием Сильвестра**

Для проверки характера найденной точки необходимо построить матрицу Гессе функции Лагранжа по переменным $x$ и $y$:

$$H = \begin{pmatrix}
\frac{\partial^2 L}{\partial x^2} & \frac{\partial^2 L}{\partial x \partial y} \\
\frac{\partial^2 L}{\partial y \partial x} & \frac{\partial^2 L}{\partial y^2}
\end{pmatrix} = \begin{pmatrix}
2 & 0 \\
0 & 2
\end{pmatrix}$$

Матрица Гессе является положительно определенной (все главные миноры положительны: $\Delta_1 = 2 > 0$, $\Delta_2 = 4 > 0$), что означает, что найденная точка является точкой локального минимума.

**Ответ:** Точка минимума: $(x^*, y^*) = (1.2, 0.2)$, минимальное значение функции: $f^* = 10.88$.
