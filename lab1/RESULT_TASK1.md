# Решение задачи линейного программирования №1

## Исходная задача

**Максимизировать:** Z = 2x₁ + 3x₂ + x₃ + 4x₄

**При условиях:**
- x₁ + x₂ + x₃ + x₄ ≤ 10
- 2x₁ + x₂ - x₃ + x₄ = 8
- x₂ + 2x₃ + x₄ ≥ 5
- x₁, x₂, x₃, x₄ ≥ 0
