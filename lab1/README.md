# Решение задач линейного программирования

**Автор:** Гельм Даниил

## Описание

Программа для решения задач линейного программирования (ЗЛП) с использованием симплекс-метода. Реализован двухфазный симплекс-метод для нахождения начального опорного решения и решения основной задачи.

## Структура проекта

- `simplex.py` - основной модуль с реализацией симплекс-метода
- `parser.py` - парсер для чтения ЗЛП из текстового файла
- `main.py` - главный файл программы
- `example1.txt`, `example2.txt`, `example3.txt` - примеры входных файлов

## Использование

```bash
python3 main.py <input_file>
```

## Формат входного файла

```
max/min: c1 x1 + c2 x2 + ... + cn xn
subject to:
a11 x1 + a12 x2 + ... + a1n xn <=/>=/= b1
a21 x1 + a22 x2 + ... + a2n xn <=/>=/= b2
...
x1 >= 0
x2 >= 0
...
```

Пример:
```
max: 3 x1 + 2 x2
subject to:
2 x1 + x2 <= 10
x1 + 2 x2 <= 8
x1 >= 0
x2 >= 0
```

## Алгоритм

1. **Преобразование в канонический вид:**
   - Ограничения `<=` дополняются слабыми переменными
   - Ограничения `>=` дополняются избыточными и искусственными переменными
   - Ограничения `=` дополняются искусственными переменными
   - Минимизация преобразуется в максимизацию (смена знака целевой функции)

2. **Двухфазный симплекс-метод:**
   - **Фаза 1:** Минимизация суммы искусственных переменных для нахождения начального опорного решения
   - **Фаза 2:** Решение исходной задачи с найденным опорным решением

3. **Симплекс-метод:**
   - Вычисление базисного решения
   - Вычисление reduced costs (оценок)
   - Проверка оптимальности
   - Выбор входящей и выходящей переменных
   - Обновление базиса
